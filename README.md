# nomadneuro.com

Marketing site for Nomad Neuroscience Ltd. Built with Vite + React + Tailwind, deployed to Netlify from the `main` branch.

## Develop

```bash
npm install
npm run dev      # local dev server
npm run build    # production build → dist/
npm run preview  # preview the production build
```

## Structure

- `index.html`, `signin.html`, `privacy.html`, `careers.html`, `404.html` — multi-page Vite entry points
- `src/main*.{jsx,js}` — page entry scripts
- `src/index.css` — Tailwind + global styles
- `public/` — static assets copied verbatim to `dist/` (favicon, images, `_headers`, `_redirects`, `robots.txt`)
- `netlify.toml` — Netlify build + sitemap config

## Deploy

Pushing to `main` triggers a Netlify build (`npm run build`, publish `dist/`). The sitemap is generated by `@netlify/plugin-sitemap`.
